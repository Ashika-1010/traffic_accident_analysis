# 🚦 Smart Traffic Accident Analysis System

A modern Python-based analytics dashboard for analyzing Indian road accident data using CustomTkinter, Pandas, and Matplotlib.

## 📌 Features

- 🔐 Login System
- ✨ Splash Screen
- 📊 Dynamic Dashboard
- 📂 Dataset Viewer
- 🔍 Search & Filter Records
- 📈 8+ Data Visualizations
- 📤 Graph Export System
- 🌙 Modern Dark UI

## 📊 Visualizations Included

- Severity Analysis
- Weather Analysis
- City Analysis
- Time Analysis
- Traffic Density Analysis
- Road Type Analysis
- Casualties Analysis
- Risk Score Analysis

## 🛠 Technologies Used

- Python
- CustomTkinter
- Pandas
- Matplotlib
